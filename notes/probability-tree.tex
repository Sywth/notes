\begin{definition}[Tree]
	A tree \(G\) is the four tuple \(G := (V,E,r,p)\) with some partial ordering \(\prec_{p}\) due to the parent function \(p : V \setminus \set{r} \to V\). We call \(r\) the root of the tree, \(V\) the set of all nodes expect \(r\) (which is also a node) and \(E \subseteq V^{2}\) the edge relation.
\end{definition}
